📘 使い方ガイド

1. このマクロの目的
Excelでの手作業を削減し、シート整形やチェック作業を自動化
入力制限・整形・一致チェックなど、業務全体の流れをサポート

2. 使用前の準備
VBAエディタ（Alt + F11）で .bas ファイルをインポート
対象ブックに「やるやら」「判定者」など必要なシートを用意

3. マクロ一覧と役割	
マクロ名				ProcessSheets	
説明				  やるやらシートを生成      	    

マクロ名				checkMacro				       	
説明				  シートを列比較し、不一致を赤塗り          	    

マクロ名				compareyaruyara_hanteiyouhi
説明				  室課・判定要否の一致確認          	    

マクロ名				compareYaruyaraColumns			
説明				  任意列のチェック		          	    

マクロ名				finalLastcheckfiles_macrosheet		
説明				  役割・Emailの整合チェック          	    

4. 実行手順（例：やるやら生成）
Alt + F8 → ProcessSheets を実行
各シートの情報が1つの「やるやら」シートにまとめられる
色付け・入力制限・フィルター・条件付き書式などが自動適用
（※スクリーンショットつきだとより親切）

5. よくある質問（FAQ）
Q：ラベルが見つからないと表示される？
→ 各シートの1行目に「A0 No.」「判定ランク」などが必要です。
Q：入力できないセルがある？
→ 一部列は保護されています。「〇」「×」などの判定後に自動解除されます。
